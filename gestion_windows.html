Administrador de Dispositivos

El Administrador de Dispositivos (devmgmt.msc) te permite ver y editar propiedades de hardware instalado, cambiar configuraciones, actualizar controladores y remover o deshabilitar dispositivos.

Actualización y Solución de Problemas

  • Windows puede identificar el tipo y función de un dispositivo pero fallar en encontrar un controlador
  • Resulta en "Dispositivo Desconocido" o tipo "genérico" con marca de exclamación amarilla
  • Los fabricantes a menudo proporcionan controladores actualizados en sus sitios web

Uso de propiedades del dispositivo para investigar controlador y revertir a versión anterior

Consola de Administración de Discos

El subsistema de disco almacena todos los datos generados por el sistema operativo y aplicaciones. Asegurar su confiabilidad y rendimiento es una tarea crítica de gestión.

Funcionalidad

La consola de Administración de Discos resume todos los discos fijos y removibles: HDDs, SSDs y unidades ópticas conectadas al sistema.

Terminología: Los términos unidades, volúmenes y particiones pueden ser confusos. Las particiones se configuran en HDDs y SSDs. Un volumen es una unidad de almacenamiento lógica para el SO.

Espacios de Almacenamiento: La función de discos dinámicos está obsoleta. Windows ahora soporta una función de RAID de software llamada Espacios de Almacenamiento para configuraciones de disco redundantes.

Tareas de Administración de Discos

Inicializar Discos

Al agregar un HDD, SSD o unidad USB sin formato, debes inicializarlo usando MBR o GPT.

Particionado

Configurar cada disco con al menos una partición. Crear nuevas particiones haciendo clic derecho en espacio sin particionar.

Formateo

Escribir un sistema de archivos, típicamente NTFS, a nuevas particiones permitiendo que Windows lea y escriba archivos.

Reparticionado

Expandir particiones existentes cuando sea necesario si hay espacio sin particionar o remover/reducir particiones.

Importante: Durante la configuración, la partición de arranque debe ser NTFS, y la partición del sistema debe ser FAT32. Las particiones del sistema o arranque no pueden ser formateadas o eliminadas.

Herramientas de Mantenimiento de Disco

Problemas Principales del Almacenamiento

Fragmentación

En un disco duro, los archivos idealmente ocupan clústeres contiguos. Con el tiempo, se fragmentan reduciendo el rendimiento de lectura.

Capacidad

Típicamente, se crean más archivos de los que se eliminan, reduciendo capacidad. El rendimiento sufre si el volumen de arranque tiene menos del 20% de espacio libre.

Daño

Los HDDs son propensos a daño físico, especialmente durante cortes de energía, llevando a archivos corruptos.

Mantenimiento Regular

El uso regular de herramientas de mantenimiento de disco, al menos mensualmente y antes de instalaciones de software.

Desfragmentador de Disco

La herramienta Desfragmentar y Optimizar Unidades mejora el rendimiento de HDD y SSD:

Para HDDs

En un HDD, desfragmentar reorganiza los datos de archivos en clústeres contiguos, reduciendo el tiempo que el controlador necesita para buscar a través del disco.

Para SSDs

En un SSD, los datos se almacenan en bloques gestionados por el controlador de la unidad. El optimizador ejecuta operaciones TRIM, permitiendo al controlador marcar datos eliminables del SO como escribibles.

Optimizar Unidades (Desfragmentador) en Windows 11

Automatización: Windows usa el Programador de Tareas para ejecutar automáticamente el optimizador de disco. Verificar regularmente cualquier problema, como ejecuciones no exitosas.

Limpieza de Disco

La herramienta Limpieza de Disco identifica archivos seguros para eliminación para liberar espacio, incluyendo aquellos en la Papelera de Reciclaje y archivos temporales.

Modo Administrador

Ejecutarlo en modo administrador con la opción "Limpiar archivos del sistema" recupera espacio de cachés como Windows Update y Defender.

Archivos que Puede Limpiar:

  • Archivos de la Papelera de Reciclaje
  • Archivos temporales
  • Cachés de Windows Update
  • Archivos de Windows Defender

Programador de Tareas

El Programador de Tareas automatiza comandos y scripts, con muchos procesos de Windows teniendo horarios predefinidos.

Características Clave

  • Desencadenadores: Las tareas pueden ser activadas por eventos, como inicio de sesión del usuario o despertar de la máquina
  • Múltiples Acciones: Cada tarea puede incluir múltiples acciones para automatización compleja
  • Registro: Toda la actividad de tareas se registra para investigar fallas
  • Organización: Las tareas pueden organizarse en carpetas para mejor gestión

Programador de Tareas mostrando una tarea de auto-actualización de Dell configurada para ejecutarse cada semana

Credenciales: Además de especificar la ruta del archivo o script y el desencadenador, debes ingresar las credenciales bajo las cuales la tarea se ejecutará. Si la cuenta de usuario carece de permisos suficientes, la tarea no se ejecutará.

Consola de Usuarios y Grupos Locales

La consola de Usuarios y Grupos Locales ofrece una interfaz avanzada para gestionar cuentas de usuario, incluyendo crear, modificar, deshabilitar, eliminar y reiniciar contraseñas.

Grupos de Seguridad

Los grupos de seguridad te permiten agrupar cuentas de usuario con permisos similares, como editar archivos en una carpeta compartida.

Grupos Predeterminados:

  • Administradores: Privilegios completos del sistema
  • Usuarios: Acceso estándar limitado
  • Invitados: Acceso muy limitado

Consola de Usuarios y Grupos Locales mostrando grupos de seguridad predeterminados

Administrador de Certificados

Un certificado digital verifica la identidad de un usuario, computadora o servicio, con validez garantizada por la autoridad de certificación (CA) emisora.

Subcarpetas Clave

  • Carpeta Personal: Almacena certificados emitidos a la cuenta de usuario
  • Autoridades de Certificación Raíz Confiables: Contiene certificados de todos los emisores confiables
  • Autoridades de Certificación Raíz de Terceros: Contiene emisores confiables de proveedores no Microsoft

Vulnerabilidades de Seguridad: Confiar en una CA insegura genera vulnerabilidades de seguridad críticas. Un certificado de CA fraudulento podría permitir que un sitio web se haga pasar por un banco legítimo.

Uso del Administrador de Certificados para ver certificados del usuario actual

Editor de Políticas de Grupo

Las herramientas GUI como Configuración y Panel de Control modifican perfiles de usuario y configuraciones del sistema almacenadas en el registro, pero muchas configuraciones del registro no son accesibles a través de estas herramientas.

Funcionalidad

El Editor de Políticas de Grupo ofrece una forma robusta de configurar estas configuraciones de Windows sin editar directamente el registro.

Beneficios en Redes Grandes: En redes grandes, las políticas de grupo aplican eficientemente configuraciones a través de múltiples computadoras, evitando configuración manual.

Uso del Editor de Políticas de Grupo para ver la política de contraseñas local

Editor de Política de Seguridad Local: Usa el editor de Política de Seguridad Local (secpol.msc) para modificar configuraciones de seguridad específicamente.

Editor del Registro

El registro de Windows es una base de datos accesible remotamente para almacenar información de configuración para el SO, dispositivos y aplicaciones.

Cinco Claves Raíz

  • HKEY_LOCAL_MACHINE (HKLM): Gestiona configuraciones del sistema
  • HKEY_USERS (HKU): Contiene configuraciones para perfiles de usuario individuales
  • HKEY_CURRENT_USER (HKCU): Subconjunto de HKEY_USERS con configuraciones para el usuario conectado
  • HKEY_CLASSES_ROOT (HKCR): Información sobre aplicaciones registradas y asociaciones de archivos
  • HKEY_CURRENT_CONFIG (HKCC): Refleja el perfil de hardware actual usado al inicio

Herramientas de Línea de Comandos

Se te requiere automatizar varias tareas rutinarias usando herramientas de línea de comandos para mejorar la eficiencia y reducir errores manuales en las operaciones de TI de la empresa.

Símbolo del Sistema

Puedes ejecutar cualquier comando desde el diálogo Ejecutar, pero para una serie de comandos o ver salida, usa el shell de comandos.

Privilegios Elevados

Algunos comandos requieren privilegios elevados. Para ejecutar como administrador, haz clic derecho en el acceso directo del Símbolo del Sistema.

Sintaxis de Comandos

Ingresar comandos en el prompt (>) usando el nombre del comando, modificadores y argumentos con sintaxis apropiada.

Sistema de Ayuda

El símbolo del sistema incluye un sistema de ayuda básico. Escribe help y presiona ENTER para ver comandos disponibles.

Elevación Requerida: Si un comando necesita elevación, verás "La operación solicitada requiere elevación." El título mostrará "Administrador: Símbolo del Sistema".

Comandos de Navegación

Comando Función Ejemplo
dir Listar archivos y subdirectorios dir /o:n (ordenar por nombre)
cd Cambiar directorio de trabajo cd C:\Users\David
cd .. Subir al directorio padre cd .. (desde Documents a David)
D: Cambiar unidad activa D: (cambia a unidad D)

Comodines: Usa caracteres comodín para caracteres no especificados. Un signo de interrogación ? representa un solo carácter no especificado. Por ejemplo, dir ????????.log lista todos los archivos .log con nombres de ocho caracteres.

Comandos de Gestión de Archivos y Discos

move y copy

Permiten transferir archivos dentro de un directorio único. Usan sintaxis de tres partes: comando Origen Destino.

robocopy

Utilidad de "copia robusta". Microsoft recomienda usar robocopy sobre xcopy para mejor manejo de nombres largos y atributos NTFS.

diskpart

Sirve como la interfaz de línea de comandos para la herramienta de Administración de Discos. Permite acciones destructivas.

chkdsk

Escanea el sistema de archivos y sectores del disco en busca de fallas e intenta reparar problemas detectados.

Precaución con diskpart: A diferencia de la herramienta de Administración de Discos, diskpart permite acciones destructivas, como eliminar volúmenes del sistema o arranque, así que úsalo con precaución.

Comandos del Sistema

Comando Función Ejemplo
shutdown /s Apagado seguro del sistema shutdown /s /t 30 (apagar en 30 segundos)
shutdown /r Reiniciar el sistema shutdown /r (reinicio suave)
sfc /scannow Verificar y restaurar archivos del sistema sfc /scannow (escaneo inmediato)
winver Información de versión de Windows winver (muestra detalles del SO)
whoami Verificar identidad del usuario actual whoami /groups (grupos del usuario)

Protección de Recursos de Windows: La utilidad sfc permite verificar y restaurar archivos del sistema desde caché si están corruptos o dañados.

Consolas Personalizadas de Microsoft Management

Una Microsoft Management Console (MMC) es un contenedor para complementos como Administrador de Dispositivos, Administración de Discos, Editor de Políticas de Grupo y Administrador de Certificados.

Personalización

  • Usa el comando `mmc` para personalizar y crear una consola con tus complementos elegidos
  • Guarda la consola como un archivo MSC en la carpeta Herramientas Administrativas
  • Agrega los complementos que necesitas para tu flujo de trabajo específico

Beneficios: Crear consolas personalizadas permite agrupar herramientas relacionadas en una sola interfaz, mejorando la eficiencia del trabajo administrativo.

Conexiones de Red Cableadas

Casi todas las conexiones de red cableadas se basan en algún tipo de Ethernet. El tipo de medio del adaptador debe coincidir con el del switch al que está conectado.

Características

  • La mayoría usa cable de alambre de cobre con conectores RJ45
  • Algunas instalaciones corporativas pueden usar cableado de fibra óptica
  • El adaptador y switch deben usar las mismas configuraciones Ethernet
  • Estas usualmente se establecen para autonegociar

Nomenclatura: Bajo Windows, cada adaptador cableado recibe un nombre. El primer adaptador se etiqueta como `Ethernet`. Los adaptadores adicionales se identifican como `Ethernet2`, `Ethernet3`, etc.

VPN y Tipos de Conexión Especiales

Red Privada Virtual (VPN)

Una red privada virtual conecta componentes y recursos de dos redes (privadas) sobre otra red (pública). Es un "túnel" seguro a través de Internet.

Red de Área Amplia Inalámbrica (WWAN)

Se refiere a usar un adaptador celular para conectar a Internet vía red del proveedor. El ancho de banda depende de las tecnologías soportadas (3G, 4G, 5G).

Configuración de Proxy

Algunas redes usan un servidor proxy para proporcionar conectividad de red. Puede mejorar tanto rendimiento como seguridad.

Conexión Medida

Los proveedores celulares pueden imponer cargos altos si se excede la asignación de datos del suscriptor. Puedes definir la red como conexión medida.

Configuración de Firewall: Cada conexión de red se rige por las configuraciones del firewall del SO local impuestas por Windows Defender Firewall. Los perfiles incluyen Privado, Público y Dominio.
