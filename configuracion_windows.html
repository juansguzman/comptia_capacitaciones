Introducción al Sistema Operativo

El sistema operativo (SO) es el software que proporciona una interfaz de usuario al hardware de la computadora y proporciona un entorno en el cual ejecutar aplicaciones de software y crear redes de computadoras.

Rol del Profesional de TI

Como representante profesional de soporte de TI, tu trabajo incluirá instalar, configurar, mantener y solucionar problemas de sistemas operativos de PC.

Conocimiento Fundamental

Necesitas entender las versiones, características, componentes y capacidades técnicas del SO.

Soporte Efectivo

Con este conocimiento, puedes proporcionar soporte efectivo para todos los tipos de entornos de sistema.

Interfaces Administrativas

Aprenderás cómo las interfaces administrativas básicas para Windows 10 y 11 pueden usarse para configurar ajustes de usuario y sistema.

Configuraciones de Usuario de Windows

Escenario de Soporte

Un usuario ha reportado que su computadora no está funcionando correctamente. Se está quejando de varios problemas:

Texto Demasiado Pequeño

El texto es muy pequeño para que puedan leerlo cómodamente.

Reloj Incorrecto

El reloj no muestra la hora correcta.

Problemas con Laptop

Problemas con la laptop que usan para trabajar remotamente.

Gestión de Energía

Cuando cierran la tapa, la computadora no se apaga como solía hacerlo, drenando la batería.

Objetivo: Ser capaz de configurar las configuraciones del sistema operativo ayudará a este usuario mientras utiliza el sistema y viaja con él.

Interfaces de Windows

Un SO está compuesto de archivos del kernel y controladores de dispositivos para interfaz con el hardware, más programas para proporcionar una interfaz de usuario y herramientas de configuración.

Evolución Histórica

Los primeros sistemas operativos para PC, como el Sistema Operativo de Disco (DOS) de Microsoft, usaban una interfaz de línea de comandos.

Interfaz Gráfica (GUI)

Windows y las aplicaciones de software para Windows se marcaron por el uso de una interfaz gráfica de usuario (GUI).

Facilidad de Uso

Esto ayudó a hacer las computadoras más fáciles de usar por personal no técnico y usuarios domésticos.

Preferencias del Cliente

El estilo de escritorio GUI favorecido por un SO particular es un factor poderoso en determinar las preferencias del cliente.

Escritorio de Windows 10

Una de las funciones principales de un SO es proporcionar una interfaz (o shell) para que el usuario configure y opere el hardware y software de la computadora.

Componentes del Escritorio

  • Menú Inicio: Se activa seleccionando el botón Inicio o presionando la tecla START o logo de Windows
  • Barra de tareas: Contiene iconos para aplicaciones abiertas y permite cambiar entre ventanas
  • Íconos de acceso directo: Para lanzar aplicaciones rápidamente
  • Área de notificación: Contiene íconos para procesos en segundo plano

Escritorio y Menú Inicio de Windows 10 (21H2)

Atajos de Teclado: Vale la pena aprender los atajos de teclado para navegar el escritorio y ventanas de programas rápidamente.

Escritorio de Windows 11

Windows 11 refresca el estilo del escritorio introduciendo una barra de tareas alineada al centro, mejor espaciado para control táctil y esquinas redondeadas.

Nuevas Características

  • Barra de tareas centrada: Nueva alineación visual más moderna
  • Mejor espaciado: Optimizado para control táctil
  • Esquinas redondeadas: Diseño más suave
  • Múltiples escritorios: Función más accesible para diferentes espacios de trabajo

Múltiples Escritorios: Permiten al usuario configurar diferentes espacios de trabajo, como un escritorio con ventanas para aplicaciones de negocios abiertas y otro con ventanas y accesos directos para aplicaciones personales y juegos.

Configuración de Windows y Panel de Control

La aplicación Configuración de Windows y el Panel de Control son las dos interfaces principales para administrar Windows.

Interfaz Descripción Uso Preferido
Configuración de Windows Interfaz habilitada para toque para gestionar Windows Interfaz administrativa preferida
Panel de Control Interfaz heredada con applets tradicionales Configuraciones avanzadas y opciones no migradas

Registro: Todos los datos de configuración de Windows se mantienen finalmente en una base de datos llamada el registro. Configuración de Windows y Panel de Control contienen páginas gráficas y applets para modificar estas configuraciones.

Aplicación Configuración de Windows

La aplicación Configuración de Windows es una interfaz habilitada para toque para gestionar Windows. Las "páginas" de opciones de configuración se dividen entre algunos encabezados principales.

Diferencias entre Versiones

  • Windows 10: Tiene una página "inicio" con todos los encabezados de configuración
  • Windows 11: No tiene página "inicio", usa íconos de navegación para moverse entre grupos de encabezados

Página principal en la aplicación Configuración de Windows 10 mostrando los encabezados de configuración de nivel superior

Panel de Control

La mayoría de las configuraciones estándar de Windows 10 y Windows 11 pueden ubicarse dentro de Configuración de Windows, pero no todas. Algunas opciones aún se configuran vía la interfaz heredada del Panel de Control.

Características

  • Cada ícono representa un applet usado para tareas de configuración
  • La mayoría de applets son agregados por Windows
  • Algunas aplicaciones de software agregan sus propios applets
  • Ejemplos: software antivirus con sus propios paneles de configuración

Panel de Control de Windows 11 mostrando todos los elementos

Configuración de Cuentas

Una cuenta de usuario controla el acceso a la computadora. Cada cuenta puede asignarse derechos o privilegios para hacer cambios de configuración del SO.

Autenticación

Una cuenta de usuario está protegida por autenticar al propietario de la cuenta usando datos conocidos solo por el propietario.

Perfil de Usuario

Cada cuenta se asocia con un perfil que contiene carpetas predeterminadas para documentos personales, imágenes, videos y música.

Cuenta Administrador

El primer usuario se configura como cuenta administrador predeterminada con privilegios para cambiar cualquier aspecto de la configuración del sistema.

Usuarios Estándar

Las cuentas adicionales usualmente se configuran como usuarios estándar con privilegios solo en su perfil.

Cuentas de Windows

Tipos de Cuentas

Tipo de Cuenta Descripción Características
Cuenta Local Puede usarse para iniciar sesión en una sola computadora únicamente Configuración local sin sincronización
Cuenta Microsoft Da acceso a servicios en la nube de Microsoft Permite inicio de sesión y sincronización de configuraciones a través de múltiples dispositivos

Configuraciones de Cuenta: La aplicación de Configuración de Cuentas se usa para gestionar información del usuario actual, agregar credenciales de inicio de sesión para otras cuentas, configurar opciones de inicio de sesión, unirse a redes de dominio, y gestionar configuraciones de sincronización.

Applet de Cuentas de Usuario

El Applet de Cuentas de Usuario en el Panel de Control es la interfaz heredada. Muchas de sus funciones están ofuscadas u ocultas y puede que no se usen fácilmente para agregar nuevas cuentas.

Funciones Principales

  • Proporciona opciones para ajustar el nombre de cuenta
  • Cambiar el nivel de privilegio entre administrador y usuario estándar
  • Cambiar configuraciones de Control de Cuentas de Usuario (UAC)

Control de Cuentas de Usuario (UAC): Es un sistema para prevenir uso no autorizado de privilegios de administrador. En el nivel de configuración predeterminado, cambiar una configuración administrativa requiere que el usuario confirme una solicitud.

Configuración de Privacidad y Seguridad

Las Configuraciones de Privacidad y Seguridad gobiernan qué datos de uso Windows tiene permitido recopilar y qué funciones de dispositivo están habilitadas y para qué aplicaciones.

Categorías Principales

  • Recopilación de datos: Permite a Microsoft procesar telemetría de uso
  • Permisos de aplicaciones: Permiten o niegan acceso a dispositivos como servicio de ubicación, cámara y micrófono
  • Configuraciones de seguridad: Permiten cambiar configuraciones de antivirus, navegador y firewall

Configuraciones generales de privacidad en Windows 11

Datos de Usuario: Los permisos de aplicaciones también controlan el acceso a datos de usuario como contactos, elementos de calendario, email y archivos.

Configuración de Escritorio

Configuración de Hora e Idioma

Las páginas de configuraciones de Hora e Idioma se usan para dos propósitos principales:

Fecha/Hora y Zona Horaria

Establecer la fecha/hora correcta y zona horaria. Mantener el PC sincronizado a una fuente de tiempo precisa es importante para procesos como autenticación y respaldo.

Opciones Regionales

Establecer opciones de región para ortografía y localización apropiadas, método de entrada de teclado y reconocimiento de voz.

Múltiples Idiomas

Opcionalmente, múltiples idiomas pueden habilitarse. El idioma activo se cambia usando un ícono en el área de notificación.

Configuración de Personalización

Permiten seleccionar y personalizar temas, que establecen la apariencia del entorno de escritorio.

Configuración de Idioma

Las configuraciones de idioma permiten configurar múltiples idiomas y métodos de entrada para adaptarse a diferentes usuarios y regiones.

Elementos de Configuración

  • Idioma de Visualización: Idioma principal de la interfaz de Windows
  • Teclados: Distribuciones de teclado para diferentes idiomas
  • Reconocimiento de Voz: Configuración para entrada por voz
  • Corrección Ortográfica: Diccionarios y reglas para diferentes idiomas

Cambio Rápido: El idioma activo se cambia usando START+SPACE o un ícono en el área de notificación.

Configuraciones de idioma

Configuración de Accesibilidad

Las configuraciones de Accesibilidad (o Facilidad de Acceso) configuran opciones de entrada y salida para adaptarse mejor a cada usuario.

Tres Grupos Principales

  • Visión: Indicadores de cursor, modos de alto contraste y filtros de color, herramienta Lupa y Narrador
  • Audición: Opciones de volumen, mezcla de sonido mono, notificaciones visuales y subtítulos
  • Interacción: Opciones de usabilidad de teclado y mouse, métodos de entrada controlados por voz y ojos

Configuraciones de accesibilidad en Windows 11

Disponibilidad: La Facilidad de Acceso puede configurarse vía Configuración o vía Panel de Control. En Windows 11, estas configuraciones se encuentran bajo el encabezado Accesibilidad.

Explorador de Archivos

La gestión de archivos es una parte crítica del uso de una computadora. En Windows, la gestión de archivos se realiza usando la aplicación Explorador de Archivos.

Funciones Principales

  • Abrir, copiar, mover, renombrar archivos y carpetas
  • Ver y eliminar archivos y carpetas
  • Navegar por la estructura de directorios
  • Acceder a ubicaciones de red

Nota: El Explorador de Archivos a menudo se refiere simplemente como "Explorer," ya que el proceso se ejecuta desde el archivo explorer.exe.

Explorador de Archivos en Windows 10

Objetos del Sistema

En Windows, el acceso a archivos de datos típicamente se media por objetos del sistema. Estos se muestran en el panel de navegación izquierdo en el Explorador de Archivos.

Objeto del Sistema Descripción
Cuenta de Usuario Contiene carpetas de datos personales pertenecientes al perfil de cuenta con sesión iniciada
OneDrive Muestra archivos y carpetas guardados en el servicio de almacenamiento en la nube (si tienes cuenta Microsoft)
Este PC Contiene carpetas personales del perfil y discos fijos y unidades de almacenamiento removibles
Red Contiene computadoras, carpetas compartidas e impresoras compartidas disponibles en la red
Papelera de Reciclaje Proporciona opción para recuperar archivos y carpetas marcados para eliminación

Unidades y Carpetas

Mientras los objetos del sistema representan áreas de almacenamiento lógicas, los archivos de datos reales se escriben a unidades de disco.

Estructura de Directorio

  • Directorio Raíz: Representado por la barra invertida (\). Ejemplo: C:\
  • Convenciones: A: unidad de disquete, C: partición principal con Windows
  • Jerarquía: Subdirectorios (carpetas) pueden contener subcarpetas y archivos

Tipos de Unidades: Una "unidad" puede ser un disco físico único, una partición en un disco, una carpeta de red compartida mapeada a una letra de unidad, o un disco removible.

Estructura de directorio típica de Windows

Archivos del Sistema

Los archivos del sistema son los archivos requeridos para que el sistema operativo funcione. El directorio raíz de una instalación típica de Windows normalmente contiene las siguientes carpetas:

Windows

La raíz del sistema, conteniendo controladores, registros, aplicaciones complementarias, archivos de sistema y configuración (notablemente el subdirectorio System32).

Program Files

Subdirectorios para software de aplicaciones instaladas. En versiones de 64 bits, se crea Program Files (x86) para aplicaciones de 32 bits.

Users

Almacenamiento para configuraciones de perfil y datos de usuarios. Cada usuario tiene una carpeta nombrada según su cuenta de usuario.

Subcarpetas del Perfil

Contiene NTUSER.DAT (datos del registro) más subcarpetas para archivos de datos personales, configuraciones de aplicaciones y archivos temporales.

Opciones del Explorador de Archivos

El applet Opciones del Explorador de Archivos en el Panel de Control gobierna cómo Explorer muestra carpetas y archivos.

Configuraciones Principales

  • Ocultar extensiones: Las extensiones de archivo pueden ocultarse para archivos conocidos
  • Archivos y carpetas ocultos: Pueden marcarse como "Ocultos" y no mostrarse por defecto
  • Archivos del sistema operativo protegidos: Archivos marcados con atributo Sistema como ocultos

Protección de Archivos: En Windows, la Protección de Archivos/Recursos previene que los usuarios (incluso usuarios administrativos) eliminen archivos del sistema.

Configuraciones generales y de vista en el diálogo Opciones del Explorador de Archivos

Configuración del Sistema

La página Configuración del Sistema en la aplicación Configuración presenta opciones para configurar dispositivos de entrada y salida, energía, escritorio remoto, notificaciones y portapapeles.

Información Disponible

  • Información clave de hardware
  • Información de versión del SO
  • Enlaces a configuraciones relacionadas
  • Accesos directos a páginas de configuración avanzada

Variables de Entorno: Las variables de entorno establecen varias rutas de archivos útiles. Por ejemplo, la variable %SYSTEMROOT% se expande a la ubicación de la carpeta Windows (C:\Windows, por defecto).

Actualización y Configuración de Seguridad

Las Configuraciones de Windows Update y Privacidad y Seguridad proporcionan una interfaz única para gestionar un entorno informático seguro y confiable.

Componentes Principales

  • Gestión de Parches: Tarea importante de mantenimiento para asegurar que las PC operen de manera confiable y segura
  • Aplicaciones de Seguridad: Detectan y bloquean amenazas al sistema y datos de la computadora
  • Actualizaciones de Características: Se lanzan periódicamente e introducen cambios a características y herramientas del SO

Windows Update

Registro de Actividad: La actividad de actualización se registra por el Visor de Eventos de Windows. Si una actualización falla en instalarse, debes verificar el registro para encontrar la causa.

Configuración de Dispositivos

La mayoría de dispositivos de hardware compatibles con Windows usan Plug and Play. Esto significa que Windows detecta automáticamente cuando se conecta un nuevo dispositivo.

Páginas de Configuración del Sistema

Contienen opciones para configurar dispositivos de Pantalla y Sonido.

Bluetooth y Dispositivos

Páginas de configuración contienen opciones para dispositivos de entrada, dispositivos de impresión/escaneo y gestión de periféricos.

Dispositivos Móviles

Configuraciones permiten vincular un smartphone a la computadora.

Administrador de Dispositivos

Proporciona una consola de gestión avanzada para gestionar dispositivos del sistema y periféricos.

Controladores de 64 bits: Al usar una edición de 64 bits de Windows, debes obtener controladores de dispositivo de 64 bits. Los controladores de 32 bits no funcionarán.

Herramientas Administrativas y Aplicaciones

Aplicaciones Basadas en la Nube

Sistemas de Email

Ejemplos incluyen Outlook Web a través del portal Microsoft 365 y cuentas Gmail del entorno Workspace de Google.

Almacenamiento

Soluciones como iCloud de Apple, Google Drive o OneDrive de Microsoft para acceso desde cualquier ubicación y dispositivo.

Herramientas de Colaboración

Permiten que múltiples usuarios trabajen juntos simultáneamente, como Microsoft Teams, Slack y Zoom.

Licenciamiento de Usuario

El licenciamiento de aplicaciones basadas en la nube puede diferir de las licencias de aplicaciones instaladas localmente.

Sincronización de Identidad: El uso de un proveedor de identidad en línea o basado en la nube puede simplificar cómo los usuarios inician sesión y acceden a recursos en un entorno empresarial.

Impacto en el Negocio: En un entorno corporativo, cualquier aplicación que se instale también debe ser soportada, considerando licenciamiento, soporte técnico, entrenamiento y impacto en operaciones.
