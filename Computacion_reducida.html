5.2 Otras Placas Reducidas

Alternativas especializadas a Raspberry Pi:

Aunque Raspberry Pi domina el mercado, existen otras SBCs con características únicas diseñadas para aplicaciones específicas. Cada una tiene sus fortalezas según el tipo de proyecto.

Por qué considerar alternativas:

  • Mayor rendimiento: CPUs más potentes o GPUs especializadas
  • Características únicas: Windows nativo, SATA, NPU para IA
  • Mejor precio/rendimiento: Para aplicaciones específicas
  • Disponibilidad: Durante escasez de Raspberry Pi
  • Soporte empresarial: Para aplicaciones comerciales

Principales competidores:

  • NVIDIA Jetson: Líder en IA embebida
  • LattePanda: Windows completo en SBC
  • ODROID: Alto rendimiento ARM
  • Orange Pi: Alternativa económica
  • BeagleBone: Enfoque industrial
  • Pine64: Open source hardware
Importante: La elección de SBC debe basarse en los requisitos específicos del proyecto, no solo en el precio o popularidad.

5.2.1 LattePanda - Windows en tu palma

¿Qué es LattePanda?

LattePanda es una serie de SBCs x86 que pueden ejecutar Windows 10/11 completo. Única en su categoría por incluir un Arduino Leonardo integrado, permitiendo control en tiempo real junto con aplicaciones Windows.

Especificaciones LattePanda 3 Delta:

Característica Especificación
CPU Intel Celeron N5105 (4 cores @ 2.9GHz)
GPU Intel UHD Graphics (24 EU)
RAM 8GB LPDDR4
Almacenamiento 64GB eMMC + slot M.2 NVMe
Arduino ATmega32U4 integrado (Leonardo compatible)
Precio $279 USD

✅ Ventajas:

  • Windows 10/11 Pro activado
  • Compatibilidad total x86
  • Arduino integrado para GPIO
  • Puede ejecutar software legacy
  • Excelente para kioscos y POS

❌ Desventajas:

  • Precio elevado vs ARM
  • Mayor consumo energético
  • Requiere enfriamiento activo
  • Comunidad más pequeña
  • Menos documentación DIY

5.2.2 NVIDIA Jetson Nano - IA en el Edge

¿Qué es Jetson Nano?

NVIDIA Jetson Nano es una SBC diseñada específicamente para aplicaciones de inteligencia artificial y visión por computadora. Incluye una GPU CUDA real con 128 cores, ideal para inferencia de redes neuronales.

Especificaciones técnicas:

Característica Especificación
GPU 128-core NVIDIA Maxwell™
CPU Quad-core ARM A57 @ 1.43 GHz
Memoria 4GB 64-bit LPDDR4
IA Performance 472 GFLOPS (FP16)
Video 4K @ 30fps encode/decode
Precio $99 USD (2GB) / $149 USD (4GB)

Frameworks soportados:

  • TensorFlow / TensorFlow Lite
  • PyTorch
  • Caffe / Caffe2
  • MXNet
  • Keras
  • ONNX

Aplicaciones típicas:

  • Detección de objetos en tiempo real
  • Reconocimiento facial
  • Robots autónomos
  • Drones inteligentes
  • Cámaras de seguridad con IA
  • Análisis de video industrial
Rendimiento IA: Jetson Nano puede ejecutar múltiples redes neuronales en paralelo. Por ejemplo: YOLOv3 a 25 FPS para detección de objetos en tiempo real.

5.2.3 ODROID - Potencia bruta ARM

¿Qué es ODROID?

ODROID (Open + Android) es una línea de SBCs de Hardkernel (Corea del Sur) conocida por ofrecer alto rendimiento a precio competitivo. Popular entre entusiastas que buscan más potencia que Raspberry Pi.

Modelo destacado: ODROID-N2+

Característica Especificación
CPU 4x Cortex-A73 @ 2.4GHz + 2x Cortex-A53 @ 2GHz
GPU Mali-G52 MP4 @ 846MHz
RAM 4GB DDR4
Ethernet Gigabit real (no compartido con USB)
USB 4x USB 3.0
Precio $79 USD

Serie ODROID popular:

  • ODROID-C4: Alternativa directa a Pi 4 ($54)
  • ODROID-HC4: NAS con 2x SATA nativo
  • ODROID-GO: Consola gaming portátil
  • ODROID-XU4: 8 cores para servidores
  • ODROID-M1: Con NPU para IA
Benchmark: ODROID-N2+ supera a Raspberry Pi 4 en un 40-50% en pruebas multi-core y tiene mejor rendimiento en emulación de juegos.

5.2.4 Arduino - Microcontrolador, NO SBC

⚠️ Aclaración importante

Arduino NO es una SBC. Es una plataforma de microcontroladores para control de hardware en tiempo real. No ejecuta sistemas operativos.

Diferencias clave: Arduino vs SBC

Característica Arduino SBC (ej: Raspberry Pi)
Sistema Operativo ❌ No tiene ✅ Linux, Windows, etc.
Tiempo real ✅ Excelente ❌ Limitado
Consumo energético ✅ Muy bajo (mW) ❌ Alto (W)
Precio ✅ $10-30 ❌ $35+
Complejidad ✅ Simple ❌ Complejo
Potencia de cómputo ❌ Limitada ✅ Alta

Cuándo usar Arduino:

  • Control de motores y servos
  • Lectura de sensores
  • Proyectos con batería
  • Control en tiempo real
  • Interfaces simples

Cuándo usar SBC:

  • Procesamiento complejo
  • Interfaces gráficas
  • Conectividad internet
  • Bases de datos
  • Multitarea
Combinación perfecta: Muchos proyectos usan Arduino para control en tiempo real + Raspberry Pi para procesamiento, comunicándose por serial o I2C.

Introducción a las Placas Reducidas (SBC)

¿Qué es una SBC?

Una Single Board Computer (SBC) es una computadora completa construida en una sola placa de circuito impreso. Incluye procesador, memoria, puertos de entrada/salida y todo lo necesario para funcionar como una computadora independiente.

Características principales:

  • Tamaño compacto: Típicamente menor a 100x100mm
  • Bajo consumo: Entre 2W y 20W de potencia
  • Costo accesible: Desde $20 hasta $200
  • Sistema completo: CPU, GPU, RAM, I/O integrados
  • Versátiles: Múltiples sistemas operativos
  • GPIO: Pines para proyectos electrónicos

Aplicaciones comunes:

  • Educación: Aprendizaje de programación
  • IoT: Dispositivos conectados
  • Servidores: Web, archivos, multimedia
  • Robótica: Control de robots y drones
  • Domótica: Automatización del hogar
  • Prototipado: Desarrollo de productos
Dato importante: El mercado global de SBC se valoró en $1.5 mil millones en 2023 y se espera que alcance $2.5 mil millones para 2026, con un crecimiento anual del 12%.

5.1 Raspberry Pi - Definición e Historia

¿Qué es Raspberry Pi?

Raspberry Pi es una serie de computadoras de placa única desarrollada por la Raspberry Pi Foundation en el Reino Unido. Su objetivo principal es promover la enseñanza de ciencias de la computación en escuelas y países en desarrollo.

Historia y evolución:

  • 2012: Lanzamiento del Model B original (256MB RAM)
  • 2013: Model A más económico
  • 2014: Model B+ con mejoras significativas
  • 2015: Pi 2 con procesador quad-core
  • 2016: Pi 3 añade WiFi y Bluetooth
  • 2019: Pi 4 con USB 3.0 y soporte 4K
  • 2023: Pi 5 con PCIe y mejor rendimiento

Impacto global:

  • Más de 45 millones de unidades vendidas
  • Presente en 170+ países
  • Millones de estudiantes beneficiados
  • Comunidad de desarrolladores activa
  • Miles de proyectos open source
  • Usado en la Estación Espacial Internacional
Fundadores: Eben Upton, Rob Mullins, Jack Lang y Alan Mycroft del Laboratorio de Computación de la Universidad de Cambridge crearon Raspberry Pi para abordar la disminución de habilidades de programación en estudiantes.

5.1.2 Diseño de Raspberry Pi

Elementos de diseño:

Componentes principales:

  • SoC (System on Chip): Procesador + GPU integrados
  • RAM: Montada sobre el SoC (PoP)
  • GPIO Header: 40 pines multipropósito
  • Puertos USB: 2-4 según modelo
  • Ethernet: Gigabit en modelos recientes
  • HDMI: 1-2 puertos para display
  • Alimentación: Micro USB o USB-C

GPIO (General Purpose I/O):

40 pines incluyen:

  • 28 pines GPIO configurables
  • Alimentación: 3.3V y 5V
  • Ground (tierra)
  • Interfaces I2C (2 canales)
  • SPI (2 chip selects)
  • UART serial
  • PWM para control de motores

Pi Zero/Zero W/Zero 2 65 × 30 mm 9g Compacto

Evolución del hardware a través de generaciones:

Capacidades de procesamiento:

  • Pi 1: ~0.04 GFLOPS
  • Pi 2: ~1.5 GFLOPS
  • Pi 3: ~3.2 GFLOPS
  • Pi 4: ~13.5 GFLOPS
  • Pi 5: ~30+ GFLOPS

Mejoras clave Pi 5:

  • CPU 2-3x más rápida que Pi 4
  • GPU con soporte Vulkan 1.2
  • Dual 4Kp60 HDMI display
  • PCIe 2.0 interface
  • RTC (reloj tiempo real) integrado
  • Botón de encendido dedicado
Rendimiento: Raspberry Pi 5 ofrece un rendimiento comparable a una PC de entrada de 2015-2016, suficiente para tareas de oficina, navegación web y desarrollo de software.

5.1.4 Sistemas Operativos para Raspberry Pi

Sistema Operativo Oficial:

Raspberry Pi OS (anteriormente Raspbian)

  • Basado en Debian Linux
  • Optimizado específicamente para hardware Pi
  • Tres versiones: Lite (sin GUI), Desktop, y Full (con software adicional)
  • Disponible en 32-bit (compatible con todos) y 64-bit (Pi 3+)
  • Incluye herramientas de programación: Python, Scratch, Java

Sistemas Linux compatibles:

  • Ubuntu: Server y Desktop (20.04/22.04 LTS)
  • Manjaro: Rolling release basado en Arch
  • Kali Linux: Para seguridad y pentesting
  • DietPi: Minimalista y optimizado
  • Fedora: Versión ARM oficial

Sistemas especializados:

  • LibreELEC: Media center con Kodi
  • RetroPie: Emulación de juegos retro
  • Home Assistant OS: Domótica
  • OpenWrt: Router/firewall
  • Windows 10 IoT Core: Para Pi 2/3
Instalación fácil: Usa Raspberry Pi Imager oficial para escribir el SO a la tarjeta SD. Permite preconfigurar WiFi, SSH, nombre de usuario y contraseña antes del primer arranque.
